<div align="center">
  <h1>Evo Browser</h1>
  <p>A personal, single-user macOS browser.</p>
</div>

Evo is a native macOS browser built on SwiftUI, AppKit, and WebKit, for one user. It is not a public product — see [CONTRIBUTING.md](CONTRIBUTING.md).

See [BUILD.md](BUILD.md) for prerequisites and the full reproducible build/test commands.
